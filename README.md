# ☕ Cofrinho

Projeto desenvolvido durante a disciplina de Programação Orientada a Objetos utilizando Java.

A aplicação simula um cofrinho virtual capaz de armazenar moedas em diferentes moedas (Real, Dólar e Euro), permitindo adicionar, remover, listar moedas e calcular o valor total convertido para Real.

---

## 🎯 Funcionalidades

* Adicionar moedas
* Remover moedas
* Listar moedas cadastradas
* Converter moedas para Real
* Tratamento de entradas inválidas
* Validação dos valores informados

---

## 📚 Conceitos Aplicados

* Programação Orientada a Objetos (POO)
* Herança
* Polimorfismo
* Classes Abstratas
* Encapsulamento
* Sobrescrita de métodos (`@Override`)
* Tratamento de exceções
* Coleções (`ArrayList`)
